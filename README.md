# HOW

- `bundle install`
- `rails db:create`
- `rails db:migrate`
- `rails db:seed`
- `cd client`
- `npm i`
- `cd ..`
- `rake dev:start`

# WHERE

- Rails Admin: `localhost:3001/admin`
- Rails API: `localhost:3001/api/v1`
- React Client: `localhost:3000`
- Dummy login available after seed:
  - u: admin@example.com
  - p: password
